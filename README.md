# Password Strength Analyzer

## Overview
Password Strength Analyzer is a Machine Learning web application developed using Flask that predicts the strength of passwords.

## Features
- Password Strength Prediction
- Password Score
- Password Statistics
- Security Suggestions
- Strong Password Generator
- Copy Password Feature

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- CSS
- Bootstrap
- JavaScript

## Machine Learning Model
- TF-IDF Vectorizer
- Logistic Regression
